# 개인정보처리방침 (Privacy Policy)

**"읽긴했니? (Read or Fake?)"** (이하 "본 확장 프로그램")은(는) 사용자의 개인정보 및 데이터를 안전하게 보호하기 위해 최선을 다하고 있습니다. 본 개인정보처리방침은 크롬 웹스토어 정책에 따라, 사용자가 본 확장 프로그램을 이용할 때 어떠한 정보를 수집, 이용, 공유하는지 명확하게 안내합니다.

## 1. 수집 및 처리하는 데이터 항목

본 확장 프로그램은 서비스의 핵심 기능을 제공하기 위해 아래의 데이터를 처리합니다.

- **웹사이트 콘텐츠**: 사용자가 검증을 위해 직접 입력하거나 파일로 업로드하는 텍스트 데이터(독후감 내용 및 책 제목, 저자 등)

*참고: 본 서비스는 이름, 전화번호, 이메일 주소, 기기 식별자 등의 개인 식별 정보(PII)를 수집하거나 요구하지 않습니다.*

## 2. 데이터의 처리 목적

수집된 데이터(웹사이트 콘텐츠)는 오직 아래의 목적을 위해서만 실시간으로 처리됩니다.

- **도서 정보 검증**: 네이버 도서 검색 API를 호출하여 해당 도서가 실제로 존재하는지 확인합니다.
- **AI 감상문 진위 분석**: 사용자가 등록한 AI API(Google Gemini 또는 OpenAI)를 활용하여 독후감이 실제로 책을 읽고 작성되었는지 진위 여부를 판별합니다.

## 3. 데이터의 보관, 저장 및 파기

본 확장 프로그램은 사용자의 데이터를 철저하게 보호하며, 외부 서버에 데이터를 무단으로 수집하거나 저장하지 않습니다.

- **분석용 텍스트 데이터**: 입력 및 업로드된 독후감 텍스트는 분석이 수행되는 즉시 과정에서만 일시적으로 사용되며, 작업이 완료된 후 어떠한 원격 서버나 외부 데이터베이스에도 보존(저장)되지 않고 파기됩니다.
- **사용자 인증 정보 (API 키)**: 사용자가 분석을 위해 설정 페이지에 입력한 외부 서비스 통합용 API 키(OpenAI, Gemini, Naver)는 오직 사용자 브라우저의 로컬 저장소(`chrome.storage`)에만 안전하게 보관됩니다. 개발자의 원격 서버로 전송되지 않습니다.

## 4. 제3자 서비스와의 데이터 공유

사용자가 요청한 AI 분석 및 도서 검색을 처리하기 위해, 입력된 텍스트 데이터는 다음과 같은 제3자 서비스 API 플랫폼으로 전송될 수 있습니다. (사용자가 직접 입력하여 연결한 API 연동 기반)

- **Naver API**: 책 제목 및 저자 정보를 통한 도서 검색 목적
- **Google Gemini API**: 텍스트(독후감) 분석 목적
- **OpenAI API**: 텍스트(독후감) 분석 목적

해당 외부 서비스로 전송된 데이터는 각 벤더사(Naver, Google, OpenAI)의 개인정보처리방침에 따라 처리 및 보호됩니다.

## 5. Chrome 확장 프로그램 권한 안내

본 확장 프로그램은 기능 구현을 위해 다음의 브라우저 권한을 사용합니다.
- **storage**: 사용자의 API 키 및 기본 설정 값을 로컬에 저장하기 위해 사용됩니다.
- **sidePanel**: 주요 UI 및 분석기 인터페이스를 브라우저 사이드 패널에 렌더링하기 위해 사용됩니다.
- **host_permissions**: 분석 및 검색 API 서버(네이버, Google, OpenAI)로 명시적인 웹 요청을 보내기 위해 사용됩니다.

## 6. 사용자의 권리 및 데이터 통제권

- 사용자는 확장 프로그램의 단축 아이콘 또는 '도구 -> 확장 프로그램'의 설정 페이지에서 언제든지 자신이 입력한 API 키 정보를 삭제할 수 있습니다.
- 확장 프로그램을 기기에서 제거(Uninstall)하면 즉시 로컬 저장소의 모든 설정 및 데이터가 브라우저에서 영구적으로 삭제됩니다.
